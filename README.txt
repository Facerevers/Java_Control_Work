Подумать над структурой класса Ноутбук для магазина техники - выделить поля и
методы. Реализовать в java.
Создать множество ноутбуков.
Написать метод, который будет запрашивать у пользователя критерий (или критерии)
фильтрации и выведет ноутбуки, отвечающие фильтру. Критерии фильтрации можно
хранить в Map. Например:
“Введите цифру, соответствующую необходимому критерию:
1 - ОЗУ
2 - Объем ЖД
3 - Операционная система
4 - Цвет …
Далее нужно запросить минимальные значения для указанных критериев - сохранить
параметры фильтрации можно также в Map.
Отфильтровать ноутбуки их первоначального множества и вывести проходящие по
условиям

Ноутбук.
Поля:
1. Производитель (laptop_manufacturer)
2. Модель (laptop_model)
3. Цвет (laptop_color)
4. Операционная система (os_name)
5. Производитель жёсткого диска (hdd_manufacturer)
6. Объём жёсткого диска (hdd_size)
7. Тип оперативной памяти (ram_type)
8. Объём оперативной памяти (ram_size)
Методы:
1. Получение Производителя (get_manufacturer)
2. Получение Модели (get_model)
3. Получение Цвета (get_color)
4. Получение Операционной системы (get_os) 
5. Получение Производителя жёсткого диска (get_hdd_manuf)
6. Получение Объёма жёсткого диска (get_hdd_size)
7. Получение Типа оперативной памяти (get_ram_type)
8. Получение объёма оперативной памяти (get_ram_size)
